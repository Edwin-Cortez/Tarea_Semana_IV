# Descripción:
Este programa en C++ simula el almacenamiento temporal de 
productos en una tienda usando una lista enlazada.

#Funcionamiento:
El programa solicita al usuario ingresar los datos de 4 
productos (código, nombre y precio). Cada producto se 
almacena en un nodo creado dinámicamente con new. Los nodos 
se enlazan en el orden en que fueron ingresados. Al final, 
el programa recorre la lista con un puntero auxiliar, 
imprime todos los productos y calcula el total acumulado 
de precios. La memoria es liberada correctamente con delete 
antes de finalizar el programa.
